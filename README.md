# Plumber Servicios SPA

Landing Page y Portafolio Web avanzado desarrollado para **Plumber Servicios SPA**, especialistas en ingeniería hidráulica, desarrollo de alcantarillado, redes de agua potable y servicios de gasfitería industrial en la Región del Biobío.

- Desarrollado por **Teo Labs** [https://www.teolabs.app/](https://www.teolabs.app/)

## 🚀 Características Principales

*   **Stack Moderno:** Construido con **Next.js 14+ (App Router)** y **React 19**, aprovechando Server Components y Acciones de Servidor para una carga instantánea y optimización SEO superior.
*   **Base de Datos en Tiempo Real:** Integración con **Supabase** (`@supabase/ssr`) para almacenar y leer el portafolio de proyectos dinámicamente.
*   **Motor SEO y OpenGraph:** Totalmente optimizado para visibilidad orgánica (Etiquetas de título dinámicas por página, estructura semántica HTML5 pura, OpenGraph para tarjetas enriquecidas en redes sociales).
*   **Whatsapp Directo Inteligente:** Enlaces generados algorítmicamente que pre-llenan el cuerpo del mensaje en WhatsApp con la URL del sitio, forzando la renderización de la tarjeta de presentación al remitente.
*   **Grid "Mobile First":** Diseño 100% responsivo basado en Vanilla CSS / Tailwind CSS, incluyendo menús ocultos tipo hamburguesa fluidos.
*   **Panel de Administración CMS:** Rutas secretas están blindadas contra rastreo de bots de Google (`noindex`).
*   **Marca Dinámica Interactiva:** Exclusiva UI de *Teo Labs* implementada en componentes bajo React con aberración cromática reaccionando al eje y del mouse.

## 🛠️ Tecnologías Empleadas

- **Frontend:** Next.js, React, Tailwind CSS, Material Symbols (Google).
- **Backend (BaaS):** Supabase (PostgreSQL + Auth + Storage).
- **Tipografía:** Oswald, Inter, Black Ops One.
- **Seguridad:** Middleware nativo The Edge, Supabase RLS (Row Level Security).
